Cách lấy Spread

  1. Mở Market Watch (Ctrl+M)
  2. Chuột phải trong bảng → chọn Spread để hiện cột spread
  3. Spread hiển thị dạng point — chia 10 nếu pair 5 digits (VD: 12 = 1.2 pips)
  4. Hoặc: Kéo chart → chuột phải → Properties → tick Show Ask line → khoảng cách Bid-Ask = spread

Cách lấy Pip Value

  1. Tự động: Calculator sẽ tự tính pip value từ tỷ giá ECB (cập nhật hàng ngày). Bạn có thể nhấn ↻ để cập nhật lại.
  2. Thủ công (MT5): Nếu broker hiển thị Tick Value trong Specification → dùng giá trị đó. Lưu ý nhân 10 nếu broker hiển thị tick value (point) thay vì pip value.
  3. Với pairs nhóm 1 (EUR/USD, GBP/USD, AUD/USD, NZD/USD) — pip value luôn = $10/lot, không cần nhập.
  4. Sai số nhỏ giữa tỷ giá ECB và tỷ giá broker là bình thường và không ảnh hưởng đáng kể đến lot size.
